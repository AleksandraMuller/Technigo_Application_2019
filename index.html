What is Node.js?

Node.js is an open-source cross-platform that enables Javascript to be used for server-side programming.

Server-side programming means that the Javascript code is executed outside of a browser on the contrary to the client-side programming that runs the code in the browser.

Node.js is built on GoogleÂ’s V8 JavaScript engine. The job of the JavaScript engine is to take in JavaScript code and compile it down to machine code which the machine can actually execute.

One of the benefits of Node.js is the possibility to create dynamic websites that run on server, which allows to tailor website content for individual users.
